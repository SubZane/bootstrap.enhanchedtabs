Tabby.js is a tiny (1.1kb) tab menu script with support for AJAX loading of content and history navigation.

It will also take advantage of hashed URLs, meaning: If you click one tab, then another and use the back button the previous tab will be selected. It takes advantage of Bootstrap markup and can very easily replace the normal Bootstrap javascript tabs.
